# 2026-06-02 시간대 진입 필터 + IRYS 제외

## 배경

전략 성과 분석 결과 현 전략은 거래할수록 손실(net EV 음수)인 구조였다.
조건을 완화하면 거래는 늘지만 손실이 커지고, 조이면 거래가 거의 발생하지 않는(라이브 24h 13,433스캔 중 2건 SELECTED) 딜레마 상태였다.

핵심 발견: **기존 진입 조건(priceChange·tradeAmount·distanceFromHigh·lastCandleBullish)은 거래 수량만 줄일 뿐 승률을 의미 있게 개선하지 못한다.** 실제로 승패를 가르는 변수는 진입 필터에 없던 **진입 시간대**와 **마켓 선별**이었다.

## 분석 방법

`backtest.py`를 **현 운영 `.env` 파라미터**(TP 2.0 / SL -1.5 / 트레일 활성 2.0·trail 1.5 / Upbit 진입값)로 재구성하고, 거래별 KST 진입 시각을 기록하도록 확장했다.

- 기간: 2025-11-23 ~ 2026-05-22 (180일), 1분봉
- 마켓: KRW 거래대금 상위 10 (PROVE, XRP, BTC, ONDO, TRAC, ETH, USDT, SUI, IRYS, DOGE)
- 비용: 수수료 0.05%/leg + 슬리피지 0.05%/leg

> ⚠️ 직전 분석은 **구 파라미터**(TP 2.5 / 트레일 1.5·0.8) 백테스트 기준이라 우량 시간대가 달랐다. 파라미터가 바뀌면 시간대 패턴도 바뀌므로 현 파라미터 재백테스트가 필수였다.

## ⚠️ 미해결 발견 — DOGE/PROVE 제외 재검토 필요

현 파라미터 + 시간 필터 기준으로 **현재 제외 목록(DOGE/PROVE/TRAC)이 오히려 손해**다:
- PROVE: +31,211 (55% 승률) — 단일 최고 마켓인데 제외 중
- DOGE: 거의 중립 (-2,817)
- 진짜 독성은 IRYS(18%)·TRAC(32%)뿐

DOGE/PROVE 제외는 **구 파라미터 시절 대규모 손실**(2026-05 분석 DOGE -81K) 때문에 도입됐다. 파라미터가 바뀐 지금은 근거가 약하다. 단, 이번 변경 범위(시간 필터 + IRYS)를 넘어서므로 **제외 목록에서 DOGE/PROVE 해제는 별도 결정으로 보류**한다.

→ 권장: 다음 검토에서 `excluded-markets`를 `KRW-TRAC,KRW-IRYS`로 축소하는 안을 실거래 데이터로 재검증.

## 한계 (과최적화 위험)

1. **시간대 화이트리스트는 in-sample 최적화** — 같은 180일 데이터에서 상위 5시간을 고름. out-of-sample 검증 전까지 과신 금지.
2. 슬리피지 0.05%/leg는 가정값. 유동 마켓은 더 낮을 수 있어 실제는 fee-after(+90,680)와 slip-after(-62,832) 사이.
3. 거래 수가 여전히 많아(1,000+/180일) 수수료 드래그가 큼 → 진입 빈도 자체를 더 줄이는 방향도 검토 필요.
4. 백테스트는 단일 포지션 가정. 실거래 포지션 상한·cooldown 미반영.
