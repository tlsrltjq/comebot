# Entry Quality Filters — 2026-05-20

## 배경

2026-05-19 손실 분석 이후 stop-loss-cooldown을 활성화했으나, 진입 조건 자체의 품질을 개선하지 않으면 동일한 패턴의 손실이 반복될 수 있다. 이번 세션에서 진입 필터 3개와 스케줄러 제한 1개를 추가했다.

## 변경 사항

### 1. 스케줄러 1회 최대 BUY 수 제한 (`max-buys-per-run=2`)

- 동일 사이클에서 여러 약한 후보를 동시에 진입하는 것을 방지
- 종목 목록은 24시간 거래대금 내림차순이므로 상위 품질 종목만 진입
- 기본값 2, 0으로 설정하면 제한 없음

### 2. 최신 캔들 최소 거래대금 필터

- KRW: 1천만 원 (`min-latest-candle-trade-amount-krw=10000000`)
- USDT: 5만 달러 (`min-latest-candle-trade-amount-usdt=50000`)
- 거래대금 증가율(상대)은 베이스가 극소인 경우 거짓 신호를 만들 수 있음
- 절대 최소값으로 유동성 하한 보장

### 3. 최근 고가 대비 현재가 거리 필터 (`max-distance-from-high-rate=2`)

- 5분 윈도우 내 고가 대비 현재 종가가 2% 초과 하락하면 진입 거부
- 피크 직후 하락 중에 진입하는 것을 차단
- `lastCandleBullish=true`와 함께 사용하면 현재 모멘텀 지속성을 추가 확인

### 4. 후보 스캔 로그 (`CandidateScanLog`)

- 모든 스캔 결과(SELECTED + SKIPPED)를 기록
- 향후 통계 분석: 필터 통과율, 교환소별/종목별 패턴
- `GET /api/candidate-scan-log?range=24h&exchange=UPBIT&decision=SELECTED`
