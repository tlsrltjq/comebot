# Project Next Steps

목표:

- 보유 포지션뿐 아니라 최근 실행 이력을 기준으로 재진입 쿨다운을 둔다.

- 기본값은 기존 동작 유지로 둔다.

## 2단계: 운영 상태 화면 정리

목표:

## 3단계: 자동 실행 결과 분석

- scheduler 실행별 FILLED, REJECTED, HOLD 비율을 비교한다.
